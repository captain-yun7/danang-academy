# 0420 MVP 스펙 (원문 정제)

> 원문: `/0420 다프 MVP 상세 개발 문서 v1.pdf` (2026-04-20, 7p)

## 1. MVP 기능 5개 상세

### A. 랜딩 페이지
- **목적**: 무료 테스트 유입 + 상담 전환
- **구성**: 메인 소개 / 무료 발음 테스트 버튼 / 무료 반배정 테스트 버튼 / 학원 소개 / 수강 신청 버튼 / 성과 지표 / 후기 / 문의 버튼

### B. 무료 발음 테스트
**흐름**: 사용자 접속 → 문장 제시 → 녹음 → STT → AI 평가 → 점수+피드백 → 상담 유도

**예시 결과**
```
발음 점수: 82
좋은 점: 속도 자연스러움
개선점: 받침 발음 보완 필요
추천 반: 중급반
```

**개발 포인트**
- 녹음 UX 단순화
- 모바일 최적화
- 결과 10초 내 출력

### C. 무료 반배정 테스트
- **목적**: 상담 전 반 자동 추천
- **구성**: 간단 객관식 문법/어휘 5문항 + 발음 테스트 1문장
- **결과**: 입문반 / 초급반 / 중급반 / 고급반

### D. QR 입출석 시스템
**흐름**: 학생 QR 발급 → 입실 스캔 → 퇴실 스캔 → DB 기록 → 관리자 대시보드 반영

**관리자 화면**: 오늘 출석 인원 / 지각자 / 미출석자 / 반별 현황

### E. 관리자 대시보드 (메인 KPI)
- 총 학생 수
- 오늘 출석률
- 무료 테스트 수
- 상담 전환 수
- 평균 발음 점수
- 반별 인원

## 2. Phase 2 기능 (오픈 후)
- 수업 후 복습 시스템: 오늘 배운 문장 자동 제공 → TTS 듣기 → 녹음 제출 → AI 평가
- 주간 리포트 (학생/학부모 전달): 출석률, 발음 향상도, 숙제 수행률, 강사 코멘트

## 3. Phase 3 기능
- TOPIK 지원: 읽기 문제, 듣기 훈련, 쓰기 평가, 작문 피드백, 예상 점수
- 작문 평가: 학생 입력 → AI 문법/논리/표현 다양성 평가 → 점수 + 수정본
